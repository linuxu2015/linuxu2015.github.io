centos6搭建openvpn

发表于 2018-03-21 | 阅读次数

软件版本

openvpn-2.4.3-1.el6.x86_64.rpm
easy-rsa-2.2.2-1.el6.noarch.rpm #推荐使用2.2的版本（搜索rpm包手动安装）
pkcs11-helper-1.11-3.el6.x86_64.rpm

网络环境

vpnserver 192.168.101.241

服务端安装

安装以上rpm包

yum install openvpn pkcs11-helper -y
yum install http://download.sipxcom.org/pub/epel/6/x86_64/easy-rsa-2.2.0-2.el6.noarch.rpm

编辑配置文件

配置证书

创建服务端证书
创建客户端证书

防火墙设置

拷贝服务器证书到相应路径

启动openvpn

需在/etc/openvpn/目录下执行命令
openvpn --config /etc/openvpn/server.conf
测试客户端登录正常后后台运行
openvpn --daemon --config /etc/openvpn/server.conf

配置windows客户端

1.安装openvpn https://openvpn.net/index.php/download/community-downloads.html
2.将客户端证书保存到openvpn安装目录中config下
3.在config目录下创建并编辑配置文件sample.ovpn

启动openvpn双击右下角openvpn图标将会自动连接，右下角openvpn图标变成绿色说明连接成功
